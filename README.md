# OasisInfobyte-Retail-Sales-EDA
Retail Sales Exploratory Data Analysis Project

## Objective
Analyze retail sales data to identify customer behavior, product performance, and sales trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Tasks
- Data Loading
- Data Cleaning
- Descriptive Statistics
- Product Analysis
- Customer Analysis
- Time Series Analysis
- Insights and Recommendations

## Key Findings
- Electronics was a top-performing category.
- Female customers represented around 51% of customers.
- Sales trends and customer behavior were identified through EDA.
